
# Dự án tiêm vaccine Covid

Dùng Remix, smart contract ở /contract/Covac.sol

Deploy qua ganache
## Run Locally

Clone the project

```bash
  git clone https://github.com/HuyMagic-12/Health-Care.git
```

Go to the project directory

```bash
  cd Health-Care
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
